# 🧮 Scientific Calculator
 
A simple and user-friendly Scientific Calculator developed to perform basic and advanced mathematical calculations.

## ✨ Features

* Addition
* Subtraction
* Multiplication
* Division
* Square root
* Power calculations
* Trigonometric functions
* 
  * Sin
  * Cos
  * Tan
* Logarithmic calculations
* Percentage calculation
* Clear and reset options
* Responsive and easy-to-use interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in a web browser.
4. Start using the Scientific Calculator.

## 📂 Project Structure

```text
Scientific-Calculator/
```

## 🎯 Purpose

The purpose of this project is to create an interactive calculator that performs both basic arithmetic operations and scientific mathematical calculations.
